why we use functions in c ?
--> Functions in C are used to:
1. Modularity: Break down complex problems into smaller, manageable pieces.
2. Reusability: Write code once and reuse it multiple times, reducing redundancy.
3. Maintainability: Easier to update and maintain code by isolating functionality.
4. Readability: Improve code organization and readability by grouping related operations.
5. Abstraction: Hide implementation details, allowing users to focus on the interface.
